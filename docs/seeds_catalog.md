# Seeds Catalog

This document lists example seeds and their intended "flavors".

- `fib_seed(9)`  
  A calm, smooth-evolving seed based on Fibonacci-like numbers.

- `themed_seed_from_word("HISTORICAL")`  
  A seed biased toward historical / lore-like behavior (purely by theme).

- High-energy seeds like `[101, 3, 7777, 45, 2, 9999, 150, 8, 560]`  
  Used for more chaotic and varied outputs (synthetic markets, event storms).
