# Contributing to Morphic Semantic Engine (MSE)

Thanks for your interest in contributing!

## Ways to Contribute

- Try the examples and report issues
- Propose new use cases
- Improve documentation
- Add tests
- Optimize the core engine
- Port the engine to other languages

## Development Setup

```bash
git clone https://github.com/your-username/morphic-semantic-engine.git
cd morphic-semantic-engine
python -m venv .venv
source .venv/bin/activate  # or .venv\Scripts\activate on Windows
pip install -r requirements.txt
```

## Running Tests

```bash
pytest
```

## Pull Request Guidelines

- Fork the repo and create a feature branch.
- Keep changes focused and documented.
- Add or update tests where it makes sense.
- Run tests before opening a PR.
- Describe the motivation and behavior of your change.

## Code of Conduct

Be kind, constructive, and respectful. Disagreement is fine; disrespect is not.
